<h1> AI-Based Bert Classifier</h1>


<h2>Description</h2>
Developed a BERT-based NLP classifier in Python using Hugging Face Transformers to categorize text as Fake News, Real News, Phishing, or Legitimate Emails. Fine-tuned in Google Colab with labeled datasets and evaluated using confusion matrices, accuracy, and F1-score visualizations.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Google Colab</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
